def simulate(...)

Simulates the system.

Simulates the system between times t_start and t_end, with the initial condition X_state. dts an array of time-steps, where dts[i] is the time-step used during step i of the simualtion.

If save_path is specified, then the result of the simulation will be outputted directly to the hard-drive as a zarr array at the given path.

Parameters

X_state : np.ndarray
    Initial condition of the system.
t_start : float
    Start time
t_end : float
    End time
dts : float, or list or array of floats
    Time steps
steps_per_save : int
    Number of simulation steps per saving the state (default -1)
save_path : str
    The path of the folder to save the output to (default "")
only_save_nodes : bool
    If True, commuter nodes will not be saved (default False)
event_times : list or array of floats
    The times at which the <code>event\_function</code> should be called (default [])
event_function : function
    The function that will be called at each event time (default None)

Returns

tuple
    A tuple <code>((node\_mappings, cnode\_mappings), ts, X\_states)</code>.
    <code>X\_states</code> is an np.ndarray of shape <code>(ts.size, N)</code> where <code>N</code> is the total
    degrees of freedom of the system. If `only_save_nodes = True`
    then <code>N</code> is just the degrees of freedom of the nodes, and excludes
    the commuting nodes. <code>ts</code> is the times corresponding to <code>X\_states</code>.
    <code>node\_mappings</code> is a dictionary with keys of form <code>(a,o,i,j)</code>,
    corresponding to age-bracket, class, home and location respectively.
    <code>node\_mappings\[a,o,i,j]</code> is the column of <code>X\_states</code> for the 
    corresponding state value. Similarly, <code>cnode\_mappings</code> is
    a dictionary with keys of the form <code>(a,o,i,j,k)</code>, corresponding
    to age-bracket, class, home, origin, destination respectively.
